# ml-sms-spam-text-classification
Machine learning project for SMS spam detection using the Naive Bayes classifier with text preprocessing and feature extraction.

## Dataset

The dataset contains SMS messages labeled as spam or ham.

The dataset used in this project is the **SMS Spam Collection Dataset**, which contains labeled SMS messages classified as spam or ham.

Dataset source:
https://archive.ics.uci.edu/ml/datasets/sms+spam+collection

The dataset consists of:
- 5,574 SMS messages
- Two classes: **spam** and **ham**

## Methodology

- Text preprocessing
- Tokenization
- Feature extraction using Bag-of-Words / TF-IDF
- Naive Bayes classification

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Numpy
- Matplotlib

## Model Evaluation

The model is evaluated using accuracy, precision, recall, and confusion matrix.


## Results

The Naive Bayes model achieved good performance in classifying SMS messages into spam and non-spam categories.

Future improvements may include deep learning models such as LSTM or transformer-based approaches.
